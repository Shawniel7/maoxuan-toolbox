这是什么

一个纯静态网页。你把当下的困惑写进去,它会匹配一到三个 毛泽东著作中的分析方法(矛盾论、实践论、实事求是……), 给你一份「原文短引 + 出处链接 + 现代译读 + 类比故事 + 行动清单」。

出发点很简单:这些著作里有一套被长期使用、经历过残酷检验的分析方法 —— 抓主要矛盾、从实践中来到实践中去、没有调查就没有发言权、持久战思维。 它们被创造时的历史任务已经远去,但作为思考工具, 它们今天仍然锋利。这个网站把它们从教科书的位置搬下来, 放到当代人会遇到的具体困惑旁边。

这不是什么

不是政治宣传,也不是鸡汤。 项目不评价毛泽东的历史功过、不讨论具体政治立场, 只处理其著作中具备普适性的方法论部分。

不是专业建议。 它不替代心理咨询、法律咨询、医疗建议、职业规划师的角色。 当你面对的问题超出了"换个思路"能解决的范围,请找合格的专业人士。

不是算命,也不是万能钥匙。 匹配结果是基于关键词和标签的粗略推荐; 真正有用的工作发生在你合上浏览器、去做那三到五条行动清单的时候。

免责声明

本项目是一个学习工具,目的是提取毛泽东著作中具有普适性的 分析方法(矛盾分析、实事求是、实践认识论等), 帮助读者思考当代个人困惑。项目 不提供政治立场、不替代专业心理 / 法律 / 医疗建议。 原文引用遵循合理使用原则,仅摘短句并标明出处,鼓励读者阅读完整原著。 引用源主要为中文马克思主义文库等公开学术资源。

数据与开源

全部思想工具数据在 data/entries.json 中。 这是一份开放的、持续增长的清单 —— 欢迎通过 PR 增补新的条目、 修正引文、添加现代类比故事。贡献指南见 CONTRIBUTING.md。

项目以 MIT 协议开源。欢迎 fork 做成你自己的"思想工具箱"—— 阳明心学版、斯多葛版、佛学禅宗版、系统动力学版 …… 方法论的谱系很长, 把它们从经典里接出来,供人当代使用,是件值得做的事。

可选 AI 模式

网站默认完全本地运行、不联网。 在设置里,你可以粘贴自己的 Anthropic API key 启用 AI 模式 —— 那之后,Claude 会基于你的具体情境生成个性化的分析。 你的 key 只保存在 localStorage, 仅被发送给 api.anthropic.com, 不会上传到这个项目的任何后端(因为这个项目没有后端)。
